# 🤖 30 Days of Streamlit: AI Edition

A 30-day challenge building AI-powered apps with Streamlit and Hugging Face.
Every app is free to run and deployable to Hugging Face Spaces — no credit card required.
Claude is introduced on Day 11 as an optional upgrade path.

---

## 🧰 Core Stack

| Tool | Purpose | Cost |
|---|---|---|
| [Streamlit](https://streamlit.io) | App UI | Free |
| [Hugging Face Inference API](https://huggingface.co/docs/api-inference) | Free open-source models | Free tier |
| [Hugging Face Spaces](https://huggingface.co/spaces) | Deploy & share apps | Free |
| [Claude API](https://anthropic.com) | More powerful LLM (Day 11+) | Paid, optional |

## 🚀 Getting Started

1. Create a free account at [huggingface.co](https://huggingface.co)
2. Get your free API token at huggingface.co/settings/tokens
3. Clone this repo and follow the daily guides

```bash
git clone https://github.com/Cybersoulja/30-Days-of-Streamlit-AI
cd 30-Days-of-Streamlit-AI/day-01
pip install -r requirements.txt
streamlit run app.py
```
